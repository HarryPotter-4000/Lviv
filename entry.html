Въезд в Украину

Ознакомиться с актуальной информацией можно на сайте Минздрава Украины:

https://moz.gov.ua/

Основная Информация для въезда в Украину:

visitukraine.today BY_UA въезд-выезд, машины, отдых

Выезд из Беларуси через наземные пункты пропуска с временным ВНЖ:

Временный вид на жительство в Украине не является причиной для выезда из Беларуси (только постоянный ВНЖ).

Официальный ответ ГПК

Основание "Лечение":

Для белорусских пограничников:

  1. Приглашение от медицинской клиники на фирменном бланке с печатью, прислано по почте, распечатано на цветном принтере
  2. Приказ Мин Здравоохранения Украины о выдаче лицензии клинике (на всякий случай)

Для украинских пограничников:

  1. ПЦР тест
  2. Страховка от ураинской страховой Обериг, купленная онлайн и распечатанная
  3. Установленное приложение Вдома, которое будет активировано после проверки пограничником введенных в приложение паспортных данных (куплен виртуальный украинский номер для получения смс, которые перенаправляются на электронную почту, куплен пакет роуминга для доступа в интернет на украинском пропускном пункте)
